설치 방법</br>
1. 제공된 파일들 (KOREAN.TRS, KOREAN01.FNT etc) 을 게임 파일이 있는 폴더 (ATLANTIS.000 파일과 같은 폴더) 에 복사합니다.</br>
2. ScummVM 2.5.0 이상 버전에서 "게임 추가(A)..." 버튼을 눌러 게임을 등록합니다.</br>
3. 게임의 언어가 "Korean"으로 인식된 것을 확인하고 OK 버튼을 누릅니다.</br>
4. "시작(S)" 버튼을 눌러 게임을 시작합니다.</br>
